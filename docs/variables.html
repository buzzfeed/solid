Using Solid Variables in Custom SCSS

By importing Solid as an SCSS partial you get access to all of Solid's SASS variables. Use these to keep your styles consistent whenever you write custom SCSS.

Typography

Used by the typography utility classes.

Text Sizes

{% highlight scss %} .my-custom-text-element { font-size: $text-1; } {% endhighlight %}
$text-1 $text-2 $text-3 $text-4 $text-5 $text-6

Line Heights

{% highlight scss %} .my-custom-text-element { line-height: $line-height-1; } {% endhighlight %}
$line-height-1 $line-height-2 $line-height-3

Font Styles

{% highlight scss %} .my-custom-text-element { font-family: $sans; font-weight: $bold; } {% endhighlight %}
$sans $slab $regular $bold

Colors

Used by the color utility classes.

Fill Colors

{% highlight scss %} .my-custom-element { background-color: $fill-blue; } {% endhighlight %}
$fill-red $fill-red-lighter $fill-pink $fill-blue $fill-yellow $fill-yellow-lighter $fill-green $fill-green-lighter $fill-orange $fill-promoted-orange $fill-gray-lighter $fill-gray $fill-gray-darker $fill-white

Social Fill Colors

{% highlight scss %} .my-custom-element { background-color: $fill-facebook; } {% endhighlight %}
$fill-facebook $fill-twitter $fill-google $fill-linkedin $fill-pinterest $fill-tumblr $fill-youtube $fill-instagram $fill-vine $fill-snapchat

Text Colors

{% highlight scss %} .my-custom-text-element { color: $text-gray; } {% endhighlight %}
$text-white $text-gray $text-gray-lighter $text-gray-lightest $text-red $text-pink $text-orange $text-promoted-orange $text-green $text-blue

Social Text Colors

{% highlight scss %} .my-custom-text-element { color: $text-facebook; } {% endhighlight %}
$text-facebook $text-twitter $text-google $text-linkedin $text-pinterest $text-tumblr $text-youtube $text-instagram

Layout

Used by the layout utility classes.

Spacing

{% highlight scss %} .my-custom-element { margin-bottom: $space-2; } {% endhighlight %}
$space-05 $space-1 $space-2 $space-3 $space-4 $space-5 $space-6

Z-Index

{% highlight scss %} .my-custom-element { z-index: $z4; } {% endhighlight %}
$z1 $z2 $z3 $z4

Borders

Used by the border utility classes.

{% highlight scss %} .my-custom-element { border: $border; border-radius: $border-radius; } {% endhighlight %}
$border $border-lighter $border-radius
