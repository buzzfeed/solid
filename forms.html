Form Labels

Use the .label class to get default margins, layout and styling for form labels. BuzzFeed labels are always written in sentence case. Additionally, use the .label__required class around an asterisk to declare a field required. If a field needs to be marked as optional, wrap "(optional)" in .label__optional.

Select Fields

Use the .select class to get default styling for select fields. Optionally, use a form label above your select if the default text is not descriptive enough, if the default is one of the valid select options, or if the selection will be saved and viewable again later.

Form Warnings

Wrap each label and form field in a parent element, then add a .form-group-warning class. Additionally, add a p below the input with a .feedback-text class to inform the user what needs to be adjusted.
